# ComfyUi-maskcolor

## 项目简介
本项目为ComfyUI自定义节点开发示例，目标是实现一个可以将遮罩边缘以指定颜色叠加在原图上的节点。该节点支持边缘颜色选择、边缘宽度调整、遮罩外扩等参数，适合图像后期处理、分割可视化等场景。

## 节点功能
- **输入**：
  - 原始图像（Image）
  - 遮罩图像（Mask）
- **输出**：
  - 遮罩边缘叠加在原图上的效果图
- **参数**：
  - 边缘颜色（可选，默认红色）
  - 边缘宽度（像素，默认3）
  - 遮罩外扩（像素，默认0）

## 使用方法
1. 将节点Python文件放入ComfyUI的`custom_nodes`目录下。
2. 启动ComfyUI，节点会自动加载。
3. 在UI中添加该节点，连接输入并调整参数。

## 依赖
- 需要`opencv-python`和`numpy`库。
